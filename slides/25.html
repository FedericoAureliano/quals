Learning Even More With Negative Examples
  • Positive message chain examples are easy to get
    • Sample by fuzzing the target system
  • Negative examples are harder but are useful
    • Should we prove that a message chain can never happen before calling it a negative example?
  • Programming-by-example tools can synthesize predicates using positive and negative examples
    • Trio is a programming-by-example tool.
  • Trio can learn that (for the ring leader election protocol)
    • for every message chain c, the head payload is the largest source label that appears in c
