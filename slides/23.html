Extending Non-Erasing Patterns with Justified Constraints
  • Let S be an input set of strings and p be a learned pattern
  • The set of justifications J(S, p) is the set of assignments μ such that ∃ s ∈ S μ(s) = p
  • A constraint is justified if it does not block any justifying assignment
    • In other words, a constraint f is justified iff ∀ μ ∈ J(S, p) μ ⊨ f
  • We consider two domain-specific constraint classes
    • Single machines: |x| = 1, where x ∈ V
    • Machine kinds: x ∈ Γ+, where Γ ⊆ Σ
  • Input strings (message chains sampled from a client-server system):
    • C1S1C1
    • C2S1C2
    • C1S2C1
  • Output pattern: XYX ∧ |X| = 1 ∧ |Y| = 1 ∧ X ∈ C+ ∧ Y ∈ S+
