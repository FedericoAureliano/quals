Message Chains Definition
The set of message chains is the smallest set such that:
  • nil is a message chain; and
  • if c is a message chain, m is a message, and s and t are identifiers, then cons(s, t, m, c) is.
Let σ1∙∙∙ σn be an execution, α be the last action taken, and let s be the address of the node that took α.
A message chain c exists in σ1∙∙∙ σn iff one of the following conditions holds:
  • c is nil
  • [Start A Message Chain] c is cons(s, t, m, nil)
    • α contains no receive actions
    • α contains send(t, m)
  • [Extend A Message Chain] c is cons(s, t, m, cons(s', s, m', c'))
    • α begins with receive(s, m')
    • α contains send(t, m)
    • cons(s', s, m', c') exists in σ1∙∙∙ σn-1
